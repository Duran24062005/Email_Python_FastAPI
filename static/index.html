API para envío de emails usando FastAPI y Python

Esta API proporciona endpoints para gestionar y enviar correos electrónicos de manera eficiente.

default

Emails

Obtiene la lista de todos los emails

Respuesta (200)

[
  {
    "id": 1,
    "recipient": "user@example.com",
    "subject": "Hello World",
    "body": "Email body text",
    "sent_at": "2024-01-01T12:00:00"
  }
]

Obtiene un email específico por ID

Parámetros

email_id (path) - ID del email

Respuesta (200)

{
  "id": 1,
  "recipient": "user@example.com",
  "subject": "Hello World",
  "body": "Email body text",
  "sent_at": "2024-01-01T12:00:00"
}

Elimina un email por ID

Parámetros

email_id (path) - ID del email a eliminar

Respuesta (200)

{
  "message": "Email deleted successfully"
}

Envía un nuevo email

Request Body (EmailCreate)

{
  "recipient": "user@example.com",
  "subject": "Email Subject",
  "body": "Email body content"
}

Respuesta (200)

{
  "message": "Email sent successfully",
  "email_id": 123
}

Actualiza un email existente

Parámetros

email_id (path) - ID del email a actualizar

Request Body (EmailUpdate)

{
  "recipient": "newemail@example.com",
  "subject": "Updated Subject",
  "body": "Updated email content"
}

Respuesta (200)

{
  "message": "Email updated successfully"
}
